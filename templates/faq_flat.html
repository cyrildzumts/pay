Comment passer une commande ?

Pour passer une commande, ajoutez les articles qui vous interessent dans le Panier.

Une fois les articles ajoutés dans le Panier, allez sur la page de votre Panier en cliquant sur l'icon du panier qui se trouve sur la barre de navigation.

Une fois sur la page de votre Panier, cliquer sur aller à la caisse. Une fois que la nouvelle page est chargée, veuillez suivre la procedure jusqu'à la fin. À la fin de ces étapes, votre commande sera passée et on vous enverra une confirmation de reception de votre commande.

comment connaître le statut de ma commande ?

Pour connaître le statut de votre commande, allez sur la page de votre compte et de là cliquer sur l'option Mes commandes.

Une nouvelle page s'ouvrira, dans cette page se trouvent toutes vos commande. Sur chaque commande listé se trouve plusieurs information, y compris le statut de votre commande.

Passée : Votre commande a été placée, mais pas encore traitée.
En Traitement : Votre commande est entrain d'être traitée.
Annulée : Votre commande a été annulée , les raisons de l'annulation peuvent être celles-ci :
  • Vous l'avez annulé
  • L'article commandé n'est plus disponible
Expédiée : Votre commande est en route pour vous être livrée
Terminée : Votre commande vous a été livrée.

Comment annuler ma commande ?
Pour annuler une commande, cliquer sur le bouton Annuler depuis la page de la commande. Notez que cette option n'est possible que lorsque la commande n'est pas encore expédiée. Si l'article est deja expediée alors vous prendriez les charges liées au transport de l'article commandé.
