  • {% trans "What is a Transfer ?" %}

    {% blocktrans %} A Transfer is transaction where a user A sends money to a User B. The User A is debited with the amount to send while the User B Account is augmented with the received amount. This Operation is not cancelable. {% endblocktrans %}

  • {% trans "What is a Payment ?" %}

    {% blocktrans %} A Payment is a trade of values from one party to another for goods or services. The mean of payment is money. The user receiving the payment must have a business account. {% endblocktrans %}

  • {% trans "What is a Service ?" %}

    {% blocktrans %} A service is offered by one or more entities in exchange for money. A service is a transaction in which no physical good is transferred from the seller to the buyer. To offer a service the seller must have a business account. {% endblocktrans %}

  • {% trans "How do I find the available services ?" %}

    {% blocktrans %} From the home Page it is possible to navigate to list of available services that can be consumed by the users. On the main top bar header, click on the Available Services link. {% endblocktrans %}
